# Employee Management System

A full-stack Employee Management System built with the **MERN Stack** following industry-standard architecture and best practices.

This project is being developed as a learning journey to understand how production-grade applications are designed, implemented, and deployed.

---

## 🚀 Tech Stack

### Frontend

* React
* Vite
* React Router
* Axios
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

### Tools

* Git & GitHub
* npm
* ESLint

---

## ✨ Planned Features

### Authentication

* User Login
* JWT Authentication
* Role-Based Access Control (Admin, HR, Employee)
* Change Password

### Dashboard

* Employee Statistics
* Department Statistics
* Recent Employees

### Employee Management

* Create Employee
* View Employee
* Update Employee
* Delete Employee
* Search Employees
* Filter Employees
* Pagination

### Department Management

* Create Department
* Update Department
* Delete Department
* View Departments

### Profile

* View Profile
* Update Profile
* Upload Profile Picture
